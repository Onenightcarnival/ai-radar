### Code w/ Claude London 2026: Rethinking how we build
- 日期：2026-05-26
- 链接：https://claude.com/blog/code-w-claude-london-2026-rethinking-how-we-build
- 摘要：Claude Blog 回顾 Code w/ Claude London 2026，并整理现场 keynote、breakout session 和 workshop 的重点。文章强调 Claude Code 与 Claude Developer Platform 的新进展，尤其是 Claude Managed Agents 支持自托管沙箱公开测试版，以及 MCP tunnels 研究预览：前者让工具执行环境运行在企业可控基础设施或托管供应商环境中，后者让 agent 访问私有网络内的 MCP server，而无需暴露公网入口。
