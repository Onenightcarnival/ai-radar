### May 6, 2026 Announcements Higher usage limits for Claude and a compute deal with SpaceX
- 链接：[https://www.anthropic.com/news/higher-limits-spacex](https://www.anthropic.com/news/higher-limits-spacex)
- 发布时间：2026-05-06
- 类型：官方更新
- 摘要：Anthropic 宣布与 SpaceX 达成算力合作，将使用其 Colossus 1 数据中心的全部算力容量；同时提高 Claude Code 和 Claude API 的使用限额，包括将 Claude Code 的五小时限额翻倍、取消 Pro/Max 高峰时段限额下调，并显著提高 Claude Opus API rate limits。

## Claude Blog

### New in Claude Managed Agents: dreaming, outcomes, and multiagent orchestration
- 链接：[https://claude.com/blog/new-in-claude-managed-agents](https://claude.com/blog/new-in-claude-managed-agents)
- 发布时间：2026-05-06
- 类型：Category
- 摘要：Claude Blog 宣布 Claude Managed Agents 新增 dreaming 研究预览，并向开发者开放 outcomes、多 agent 编排和 webhooks。Dreaming 会定期复盘 agent 会话和 memory，提取跨会话模式并整理记忆；outcomes 允许开发者定义成功标准，让独立 grader 评估并推动 agent 自我修正；多 agent 编排则让 lead agent 拆解任务并协调多个 specialist 并行工作。
- 关注点：这是面向 agent 工程能力的一次集中更新，重点从“能完成任务”推进到“能长期学习、按质量标准自检、并行处理复杂工作流”。对开发者来说，Managed Agents 的评估闭环、记忆维护、可追踪编排和 webhook 通知都更接近生产级 agent harness。
