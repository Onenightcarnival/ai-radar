- 链接：[https://www.anthropic.com/news/redeploying-fable-5](https://www.anthropic.com/news/redeploying-fable-5)
- 发布时间：2026-06-30

### Claude Science, an AI workbench for scientists, is now available
- 链接：[https://www.anthropic.com/news/claude-science-ai-workbench](https://www.anthropic.com/news/claude-science-ai-workbench)
- 发布时间：2026-06-30
- 类型：Announcements
- 摘要：Anthropic 发布 Claude Science，定位为面向科研人员的 AI workbench。它把科研常用工具和软件包整合到一个环境中，支持可审计的产物、灵活计算资源、文献分析、多步骤研究、图表和论文草稿迭代，并提供 60 多个面向生命科学等领域的 skills 和 connectors。
- 关注点：这是 Anthropic 把 Claude、MCP、skills 和 agent 工作流推进到科研场景的一次产品化落地，重点不只是模型能力，而是把工具链、可复现性和审阅机制一起打包给科研团队。

### Introducing Claude Sonnet 5
- 链接：[https://www.anthropic.com/news/claude-sonnet-5](https://www.anthropic.com/news/claude-sonnet-5)
- 发布时间：2026-06-30
- 类型：Product
- 摘要：Anthropic 发布 Claude Sonnet 5，称其是目前最具 agent 能力的 Sonnet 模型，在编程、工具使用、推理和知识工作上较 Sonnet 4.6 提升，并接近 Opus 4.8 的部分能力但价格更低。该模型已面向所有套餐开放，也是 Free 和 Pro 的默认模型，并可通过 Claude Code 和 Claude API 使用。
- 关注点：这是 Sonnet 档位的一次关键升级，直接影响 Claude Code、API 和日常专业工作流的默认体验；对开发者尤其需要关注新模型 ID、价格窗口和 agent 任务能力变化。
