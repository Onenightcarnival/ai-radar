## Anthropic Research

### Agentic coding and persistent returns to expertise
- 链接：[https://www.anthropic.com/research/claude-code-expertise](https://www.anthropic.com/research/claude-code-expertise)
- 发布时间：2026-06-16
- 类型：经济影响与开发者平台
- 摘要：Anthropic 基于 2025 年 10 月至 2026 年 4 月约 40 万个 Claude Code 交互会话，分析 agentic coding 中的人机分工、用户专业度和任务成功率。报告认为，用户通常决定“做什么”，Claude 更多决定“怎么做”；领域专业度越高，Claude 每次指令能承担的动作越多，任务也更容易成功。
- 关注点：这篇研究把 Claude Code 从“开发工具使用情况”推进到“知识工作如何被 agent 重组”的证据层面，尤其值得关注非软件职业也能完成代码任务、任务价值上升、以及领域知识仍然决定产出质量这几个信号。
