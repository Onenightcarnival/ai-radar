### KPMG integrates Claude across its core business and workforce of more than 276,000 in strategic alliance
- 日期：2026-05-19
- 链接：https://www.anthropic.com/news/anthropic-kpmg
- 摘要：Anthropic 与 KPMG 宣布全球战略联盟。KPMG 将把 Claude 嵌入 Digital Gateway 平台，面向税务和法律客户推出新工具，并向全球 276,000 多名员工开放 Claude。双方还会围绕私募股权客户共同开发 Claude 驱动产品，并在网络安全、业务现代化和 Claude Code 相关场景中合作。

## Claude Blog

### New in Claude Managed Agents: self-hosted sandboxes and MCP tunnels
- 日期：2026-05-19
- 链接：https://claude.com/blog/claude-managed-agents-updates
- 摘要：Claude Blog 宣布 Claude Managed Agents 支持自托管 sandbox 和 MCP tunnels。自托管 sandbox 进入 public beta，允许企业把 agent 工具执行放在自己的基础设施或 Cloudflare、Daytona、Modal、Vercel 等托管环境中；MCP tunnels 处于 research preview，可让 agent 访问私有网络中的 MCP server，而不需要暴露公网入口。
- 关注点：这是今天最直接的开发者和企业平台更新。它把 Managed Agents 的 orchestration 留在 Anthropic 侧，同时把文件、仓库、运行时、网络策略和私有工具访问放回企业边界内，重点解决 agent 进入生产环境时的安全、审计、隔离和私有数据连接问题。
