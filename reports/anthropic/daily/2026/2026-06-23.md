- 链接：[https://www.anthropic.com/news/introducing-claude-tag](https://www.anthropic.com/news/introducing-claude-tag)
- 发布时间：2026-06-23
- 类型：Product
- 摘要：Anthropic 发布 Claude Tag beta，先从 Slack 场景开始，让团队成员可以在频道里 `@Claude` 分派任务。管理员可以限定 Claude 可访问的频道、工具、数据和代码库；Claude 会在 Slack 线程中拆解并执行任务，也能基于授权范围记住相关上下文、异步推进工作。该功能面向 Claude Enterprise 和 Team 客户开放，并替代原有 Claude in Slack 应用。
- 关注点：这是 Claude Code / Claude Cowork 从个人工作流走向团队协作界面的产品化动作，重点在 Slack、权限边界、组织级审计和异步 agent 工作模式。
