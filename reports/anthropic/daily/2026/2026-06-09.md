## Anthropic News

### [Claude Fable 5 and Claude Mythos 5](https://www.anthropic.com/news/claude-fable-5-mythos-5)
- 日期：2026-06-09
- 摘要：Anthropic 宣布推出 Claude Fable 5 和 Claude Mythos 5。Fable 5 是面向一般用户开放的 Mythos-class 模型，官方称其在软件工程、知识工作、视觉、科学研究等多类任务上超过此前公开可用的 Claude 模型；Mythos 5 使用同一底层模型，但面向少量网络防御者和基础设施伙伴开放更少限制的能力。文章同时说明了新的安全分类器、对部分网络安全、生物化学和模型蒸馏相关请求的回退机制、30 天数据保留政策，以及 Fable 5 的订阅与 API 可用性。
- 关注点：这是一条模型与安全治理同时推进的重大公告。值得关注的不是单纯的能力指标，而是 Anthropic 把“更强模型开放给普通用户”和“高风险能力通过 Project Glasswing、trusted access program 与分类器分层管理”绑定在一起。对开发者和企业用户来说，Fable 5 进入 Claude API 和企业用量计划，意味着更长任务、更复杂代码迁移和知识工作场景可能获得新上限；对安全和研究观察者来说，Mythos 5 的受控开放、误拦截率、回退到 Opus 4.8 以及新增数据保留政策，是判断 Anthropic 如何处理前沿能力风险的核心线索。

## Claude Blog

### [New in Claude Managed Agents: run agents on a schedule and store environment variables in vaults](https://claude.com/blog/whats-new-in-claude-managed-agents)
- 日期：2026-06-09
- 摘要：Claude Blog 宣布 Claude Managed Agents 的两项公开 beta 能力：一是让 agent deployment 按计划自动运行，可用于每日摘要、夜间数据同步、周期性合规扫描等重复任务；二是将 vaults 扩展到环境变量，让 agent 沙箱里的 CLI 和其他工具可以安全调用外部服务。文章强调，agent 只能拿到占位符，真实密钥在网络边界按允许域名注入，并可在 vault 中集中轮换。
- 关注点：这篇文章把 Managed Agents 从“人工触发的 agent 运行环境”推进到“可被运营的自动化平台”。定时 deployment 解决重复执行问题，环境变量 vaults 则解决 CLI、MCP、浏览器工具和企业私有 API 的凭证接入问题。对构建长期 agent 工作流的团队来说，关键变化是 Claude 平台开始提供调度、凭证隔离、工具访问和按需触发这些生产化能力，而不只是模型调用本身。
