### Introducing the Claude apps gateway for Amazon Bedrock and Google Cloud
- 链接：[https://claude.com/blog/introducing-the-claude-apps-gateway](https://claude.com/blog/introducing-the-claude-apps-gateway)
- 发布时间：2026-06-29
- 类型：Category
- 摘要：Claude apps gateway 面向使用 Amazon Bedrock 和 Google Cloud 访问 Claude Code 的企业团队，提供企业 SSO 登录、集中策略执行、基于角色的访问控制、按用户成本归因和支出上限。部署形态是运行在客户基础设施上的无状态容器。
- 关注点：这是 Claude Code 进入企业云环境时的治理补位，重点解决身份、权限、成本归因和预算控制，而不是单纯增加模型入口。

### Claude in Microsoft Foundry is now generally available
- 链接：[https://claude.com/blog/claude-in-microsoft-foundry](https://claude.com/blog/claude-in-microsoft-foundry)
- 发布时间：2026-06-29
- 类型：Category
- 摘要：Anthropic 宣布 Claude 模型已在 Microsoft Foundry 中正式可用，并由 Azure 托管，企业和开发者可以通过 Microsoft 的 AI 开发平台调用 Claude。
- 关注点：这扩大了 Claude 在主流企业云和开发者平台里的可获得性，尤其适合已经使用 Azure/Microsoft Foundry 的团队评估 Claude 集成。
