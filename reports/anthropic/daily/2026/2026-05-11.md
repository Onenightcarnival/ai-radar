### Agent view in Claude Code
- 链接：[https://claude.com/blog/agent-view-in-claude-code](https://claude.com/blog/agent-view-in-claude-code)
- 发布时间：2026-05-11
- 类型：Product announcements
- 摘要：Claude Code 新增 agent view，用一个界面管理多个 Claude Code 会话。用户可以从任意会话按左箭头进入，或在终端运行 `claude agents`；列表会显示会话状态、是否等待输入、最近响应和最近交互时间。它还支持从列表中预览会话、直接回复等待决策的任务，并通过 `/bg` 或 `claude --bg [task]` 把任务送到后台。
- 关注点：这是一条 Claude Code 的工程与开发者平台信号，重点在于 Anthropic 正把多 agent 并行工作从“多个终端窗口”收敛为可扫描、可切换、可接管的 CLI 工作台。对重度 Claude Code 用户来说，它直接影响长任务管理、并行探索和 PR/后台任务巡检。

### Introducing the Claude Platform on AWS
- 链接：[https://claude.com/blog/claude-platform-on-aws](https://claude.com/blog/claude-platform-on-aws)
- 发布时间：2026-05-11
- 类型：Product announcements
- 摘要：Claude Platform on AWS 正式可用，AWS 客户可以通过 AWS 身份认证、CloudTrail 审计和统一 AWS 账单访问 Claude Platform 功能。官方说明，该平台提供原生 Claude API 功能，包括 Claude Managed Agents、code execution、web search、web fetch、Files API、Skills、MCP connector、prompt caching、citations、batch processing 和 Claude Console；Claude Opus 4.7、Sonnet 4.6、Haiku 4.5 也可用。
- 关注点：这是一条平台和企业采用信号。它把 Anthropic 原生平台能力接入 AWS 采购、权限和审计体系，适合已有 AWS 云运营模型、但又需要 Claude API 新功能同步上线的团队；同时官方也明确区分了 Claude Platform on AWS 与 Amazon Bedrock 在服务运营和数据处理边界上的差异。
