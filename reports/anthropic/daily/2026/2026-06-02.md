### Expanding Project Glasswing
- 日期：2026-06-02
- 链接：https://www.anthropic.com/news/expanding-project-glasswing
- 摘要：Anthropic 宣布扩大 Project Glasswing，把合作范围延伸到约 150 家新组织。这批组织来自 15 个以上国家，覆盖电力、水务、医疗、通信、硬件、软件供应商、非营利组织和关键开源维护方等领域。公告称，早期 Project Glasswing 伙伴已用 Claude Mythos Preview 大规模扫描代码库并发现大量高危或严重漏洞，下一步重点会从发现漏洞继续推进到披露、修复和补丁部署。

### A harness for every task: dynamic workflows in Claude Code
- 日期：2026-06-02
- 链接：https://claude.com/blog/a-harness-for-every-task-dynamic-workflows-in-claude-code
- 摘要：Claude Blog 介绍 Claude Code dynamic workflows 的工作方式和使用模式。文章称 Claude Code 现在可以按任务动态编写 JavaScript workflow，调用 subagents、选择模型、决定是否使用独立 worktree，并在中断后继续恢复。文章重点解释了 classify-and-act、fan-out-and-synthesize、adversarial verification、generate-and-filter、tournament、loop until done 等模式，并给出迁移重构、深度研究、事实核查、排序、规则遵守和根因分析等场景。
- 关注点：这是 Claude Code 的工程实践信号。Dynamic workflows 的价值在于把长周期、并行、高验证成本的任务从单一上下文里拆出来，让不同 subagents 分别执行和校验，降低自我偏好、目标漂移和中途停止的风险。官方也提醒该能力更适合高价值复杂任务，因为会消耗更多 token，最佳实践仍在形成。
