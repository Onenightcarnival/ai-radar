### Secure access to the Claude Platform with Workload Identity Federation
- 链接：[https://claude.com/blog/workload-identity-federation](https://claude.com/blog/workload-identity-federation)
- 发布时间：2026-06-17
- 类型：开发者平台与安全
- 摘要：Claude Platform 的 Workload Identity Federation（WIF）正式可用，用请求时签发的短期、带作用域凭证替代静态 API key。官方说明 WIF 兼容 OIDC 身份提供方，并覆盖 Claude API 端点。
- 关注点：这是企业级平台安全能力补强，重点不在模型能力本身，而在降低长期 API key 泄露风险，让生产工作负载更容易接入现有身份治理和权限边界。

### Claude Design now stays on brand for daily work
- 链接：[https://claude.com/blog/claude-design-stays-on-brand-for-daily-work](https://claude.com/blog/claude-design-stays-on-brand-for-daily-work)
- 发布时间：2026-06-17
- 类型：产品能力与设计工作流
- 摘要：Claude Design 更新后可以在不同项目中遵循团队设计系统，并与 Claude Code 更顺畅协作；用户还能在画布上直接编辑，并连接更多日常使用工具。入口也被放到 Claude 桌面应用侧边栏，同时可通过 claude.ai/design 访问。
- 关注点：这表明 Claude 正在把设计、编码和日常协作进一步连成产品工作流，而不是只停留在对话式生成。对产品和设计团队来说，关键变化是品牌一致性、画布编辑和 Claude Code 协作。
