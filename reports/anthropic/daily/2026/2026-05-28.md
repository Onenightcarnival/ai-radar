## Anthropic News

### Introducing Claude Opus 4.8
- 日期：2026-05-28
- 链接：https://www.anthropic.com/news/claude-opus-4-8
- 摘要：Anthropic 发布 Claude Opus 4.8，称新版本在 coding、agentic tasks、reasoning 和专业工作任务上较 Opus 4.7 有改进，并保持 Opus 4.7 的常规价格。公告同时列出几项配套更新：claude.ai 和 Claude Cowork 增加 effort control，Claude Code 推出 dynamic workflows research preview，Messages API 支持在 messages array 中加入 system entries，便于开发者在不中断 prompt cache 的情况下更新 agent 指令或环境上下文。
- 关注点：这是模型与开发者平台信号。Opus 4.8 的发布不只是单一模型升级，还和 Claude Code 长程工作流、effort control、API harness 控制一起推出，说明 Anthropic 正在把高能力模型、可调推理成本和大规模 agent 编排打包成同一轮产品能力更新。

### Anthropic raises $65B in Series H funding at $965B post-money valuation
- 日期：2026-05-28
- 链接：https://www.anthropic.com/news/series-h
- 摘要：Anthropic 宣布完成 650 亿美元 Series H 融资，投后估值 9650 亿美元。公告称本轮融资由 Altimeter Capital、Dragoneer、Greenoaks 和 Sequoia Capital 领投，公司近期企业采用持续增长，run-rate revenue 本月早些时候超过 470 亿美元。Anthropic 表示，这笔资金将用于推进安全和可解释性研究、扩展算力以满足 Claude 需求，并扩大客户依赖的产品和合作伙伴体系。
- 关注点：这是一条商业与基础设施信号。公告把融资用途直接绑定到研究、安全、算力和产品扩张，并提到 Claude Code、Claude Cowork、AWS、Google、Broadcom、SpaceX 等能力和基础设施合作，说明 Anthropic 正在同时扩大模型训练推理供给、企业采用和开发者产品线。

## Claude Blog

### Introducing dynamic workflows in Claude Code
- 日期：2026-05-28
- 链接：https://claude.com/blog/introducing-dynamic-workflows-in-claude-code
- 摘要：Claude Blog 发布 Claude Code dynamic workflows research preview。该能力让 Claude 针对复杂任务动态写 orchestration scripts，启动数十到数百个并行 subagents，并在结果交付前进行检查。文章给出的场景包括跨代码库 bug hunt、性能优化审计、安全审计、大规模迁移、现代化改造，以及需要多次独立验证的高风险任务。Dynamic workflows 已在 Claude Code CLI、Desktop、VS Code extension、Claude API、Amazon Bedrock、Vertex AI 和 Microsoft Foundry 上提供，面向 Max、Team、Enterprise 及相关平台用户开放或可由管理员启用。
- 关注点：这是今天最重要的 Claude Code 更新。它把 Claude Code 的工作方式从单个 agent 或少量 subagents，推进到由 Claude 自行规划、拆分、并行执行、对抗验证和汇总的长周期 workflow。对工程团队来说，价值不只在生成更多代码，而在让大规模审计、迁移和验证任务有更明确的编排层；同时也要注意官方提醒，dynamic workflows 会显著消耗更多 token，首次触发时需要确认，组织管理员也可以通过 managed settings 控制启用。
