- 链接：[https://www.anthropic.com/news/economic-futures-research-fund-agenda](https://www.anthropic.com/news/economic-futures-research-fund-agenda)
- 发布时间：2026-07-22
- 类型：经济影响 / 研究资助
- 摘要：Anthropic 公布 Economic Futures Research Fund 的研究议程，并承诺投入 2 亿美元支持外部研究，重点评估社会如何为 AI 的经济影响做准备。基金优先关注五个方向：企业和工作场所层面的 AI 影响、AI 驱动转型中的个人支持、面向 AI 失业风险的收入支持现代化、在冲击到来前让劳动者分享 AI 增长收益，以及公共投资证据建设。

### How Outtake built a cyber investigator on Claude
- 链接：[https://claude.com/blog/how-outtake-built-a-cyber-investigator-on-claude](https://claude.com/blog/how-outtake-built-a-cyber-investigator-on-claude)
- 发布时间：2026-07-22
- 类型：Agents / Claude Code / Cybersecurity
- 摘要：Claude Blog 介绍 Outtake 如何用 Claude Code 和 Claude Agent SDK 构建 Recon Agent，一个可长时间运行的自主网络安全调查 agent。该 agent 会从仿冒页面出发，收集证据、追踪关联基础设施、绘制攻击网络图，并生成调查过程、威胁行为者画像和时间线；文章还披露 Outtake 通过 eval、生产级 harness、文件系统和 bash 等开放工具来支持长任务执行。
- 关注点：这篇的核心信号是 agent 应用正在从“执行单步工具”走向可持续数十分钟到数小时的行业工作流。对开发者来说，关键不是把所有行为写进 prompt，而是把稳定规则放进 orchestration 和 guardrails，把判断空间留给 agent，同时用 eval 和沙箱控制质量与风险。

### Building verification loops in Claude Code with skills
- 链接：[https://claude.com/blog/building-verification-loops-in-claude-code-with-skills](https://claude.com/blog/building-verification-loops-in-claude-code-with-skills)
- 发布时间：2026-07-22
- 类型：Claude Code / Skills / Engineering
- 摘要：Claude Blog 讲解如何把人工检查步骤写成 Claude Code skills，从而形成 verification loops。文章先列出内置验证方式，包括 `/verify` skill、工具链、Code Review、GitHub Actions、spec validation 和 Claude Managed Agents rubrics；随后建议把反复手动执行的检查用自然语言沉淀为 skill，并按 standalone、embedded、chained 或 PR 触发方式接入工作流。
